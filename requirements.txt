# NOTE on Ollama: the python client is installed here, but Ollama also needs a
# running local server + a pulled model (e.g. `ollama run llama3.2`). On a
# personal PC that is a one-time setup. In a headless/cloud sandbox there is no
# Ollama daemon, so the app detects this and falls back to deterministic,
# template-based planning/summarization. The fallback is surfaced in the UI.
#
# NOTE on embeddings: ChromaDB ships an ONNX all-MiniLM-L6-v2 embedder that needs
# a one-time model download. If sentence-transformers/torch or network are
# unavailable, retrieval falls back to a deterministic hashing embedder so the
# demo still runs offline. The active embedder is shown in Trust details.
